#### Shader中的字节对齐
对于shader中的数据结构，有不同的对齐要求，对于32位而言
float，bool等 : 4字节对齐
vec2 : 8字节对齐
vec3，vec4: 16字节对齐
在c++中可以指定对齐方式：
```
struct Uniform{
	alignas(16) vec3;
}
```
